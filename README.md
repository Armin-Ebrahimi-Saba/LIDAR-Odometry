# LIDAR Odometry

This repository explores LiDAR odometry / SLAM approaches. The `master` branch
is intentionally kept minimal — the actual implementations live in the
branches below.

## Branches

- [`KISS-ICP`](../../tree/KISS-ICP) — LiDAR odometry pipeline built on KISS-ICP.
- [`GLIM_ROS2_CPU`](../../tree/GLIM_ROS2_CPU) — GLIM-based SLAM (ROS2, CPU).
- [`LIO_SAM_6AXIS`](../../tree/LIO_SAM_6AXIS) — LIO-SAM with 6-axis IMU support.

Check out the relevant branch to work with a specific approach.
